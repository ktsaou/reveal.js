FireHOL

a Linux firewall for humans...

firehol.org
first release 2002, latest release 2017
 
FireHOL is a language to express firewalls.

  • human friendly, even with huge firewalls
  • scriptable, the config is a BASH script
  • supports IPSETs
  • supports SYNPROXY (DDoS protection)
  • creates iptables load balancers
  • creates dynamic firewalls, like auto-blacklisting on portscan, enable service after knock, etc.
  • can load-balance multiple WANs


internet="ppp+"
lan="eth0"
admins="example.firehol.org"

masquerade ${internet}

interface ${lan} home
    policy accept

interface ${internet} world
    policy drop
    protection strong

    server ping accept
    server smtp accept
    server http accept
    server ftp  accept
    server ssh  accept src ${admins}

    client all  accept

router world2home inface ${internet} outface ${lan}
    policy reject
    protection strong

    client all accept

FireQOS

a Linux traffic shapper (and still the only one)

firehol.org
first release 2013, latest release 2017
 
FireQOS is a language to express traffic shapping.

  • human friendly
  • scriptable, the config is a BASH script
  • can shape inbound and outbound traffic
  • supports encapsulation overheads calculation (e.g. ADSL/ATM overheads)
  • supports virtually unlimited nesting of classes, even with link emulation
  • can tcpdump the contents of any class 


interface eth2 world bidirectional rate 100Mbit

    class critical commit 2Mbit
            client dns
            client ntp

    class interactive commit 10Mbit
            match icmp limit 1Mbit

            server ssh
            client ssh
    
    class web_server commit 70%
            server http

    class bulk commit 5Mbit
            server smtp
            client smtp
            server rsync
            client rsync

    class default
            # the rest of the traffic
            # ends up, here

All Cybercrime IP feeds

update-ipsets and iprange

iplists.firehol.org
first release 2015, latest release 5 mins ago (it's a bot)

  • update-ipsets is a bot to download, process and analyze IPSETs.
     
    It downloads IP Feeds from their maintainers, it processes them to extract IPs, monitors the age of the IPs listed, detects their retention policy and checks how they correlate or overlap
     
  • iprange is a high performance ipsets processor (it applies sets theory operations to IPSETs). It is really very fast!
     
  • The site... well it a monitoring console. update-ipsets generates static JSON files every few minutes. These files are uploaded to the site and are rendered on charts, maps and tables.

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the page background.

State Events

Additionally custom events can be triggered on a per slide basis by binding to the data-state name.

	console.log( '"customevent" has fired' );
